## Custom Slide

Customize the slide style and background

``` markdown
{{</* slide background-image="/media/boards.jpg" */>}}
{{</* slide background-color="#0000FF" */>}}
{{</* slide class="my-style" */>}}
```

------------------------------------------------------------------------

## Custom CSS Example

Let's make headers navy colored.

Create `assets/css/reveal_custom.css` with:

``` css
.reveal section h1,
.reveal section h2,
.reveal section h3 {
  color: navy;
}
```
